This project is a simple breakout game that was built with SDL2.\hypertarget{index_Instructions}{}\doxysection{Instructions}\label{index_Instructions}

\begin{DoxyEnumerate}
\item This project was built on Windows\+OS.
\item You have to have SDL2, SDL\+\_\+image, SDL\+\_\+ttf, SDL\+\_\+mixer loaded in your environment.
\end{DoxyEnumerate}\hypertarget{index_Game}{}\doxysection{Commands}\label{index_Game}

\begin{DoxyEnumerate}
\item Use A/D key or Left/\+Right Arrow key to control your paddle(move left or right).
\item Press L key to start the game(shoot the ball).
\item Press P key to pause/unpause the game.
\item When player loses current level, press R key to restart the game(start from level one again).
\item When player wins current level, press N key to load and play next game(\+If no more levels, then restart from level one again).
\item Press Q key to quit the game. 
\end{DoxyEnumerate}
